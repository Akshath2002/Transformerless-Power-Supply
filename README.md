# Transformerless 5V Power Supply PCB

## Overview

This project is a custom-designed transformerless AC-to-DC power supply PCB developed using **KiCad 10.0**. The circuit converts AC mains voltage into a regulated 5V DC output suitable for low-power embedded and IoT applications.

The PCB was designed from schematic capture through final layout, including component selection, footprint assignment, routing, ground plane implementation, and 3D verification.

## Features

* AC mains input
* Capacitive dropper power supply
* Full-wave bridge rectifier using 1N4007 diodes
* 7805 voltage regulation
* Input surge and filtering components
* LED power indication
* Ground plane for improved noise performance
* Through-hole component design
* Two-layer PCB layout
* Designed using KiCad 10.0

## Design Workflow

### Schematic Design
